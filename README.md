# GitHubSample 프로젝트 요약

이 프로젝트는 VS Code 환경에서 GitHub 저장소를 생성하고, 로컬 소스를 동기화하는 과정을 테스트하기 위한 예제입니다.

## 주요 과정

1. **로컬 프로젝트 생성**: Python Hello World 소스(`hello.py`)를 작성
2. **Git 초기화**: `git init`으로 로컬 저장소 생성
3. **GitHub 인증**: `gh auth login`으로 GitHub CLI 인증
4. **원격 저장소 생성**: `gh repo create`로 Public 저장소 및 Python .gitignore 추가
5. **커밋 및 푸시**: `git add .`, `git commit`, `git push`로 main 브랜치에 반영
6. **README.md 작성**: 프로젝트 목적과 과정을 요약
